# Stellar - The Ultimate ToDo Experience 🚀✨

Stellar is not just another ToDo application; it's a meticulously crafted digital experience designed to bring clarity, focus, and accomplishment to your life. Inspired by the design principles of Apple and the functionality of apps like Linear and Notion, Stellar combines a breathtaking "WOW" factor UI/UX with powerful, intuitive features.

Every interaction is designed to be fluid, satisfying, and beautiful, turning the mundane task of managing your to-do list into a delightful experience.

## ✨ Features

Stellar is packed with features designed for power, flexibility, and a premium user experience.

*   **Spectacular Landing Page**: A visually stunning entry point with an animated gradient mesh background and a clean, modern design.
*   **Secure Authentication**: A beautiful, glassmorphism-style modal for seamless login and signup.
*   **Multi-View Task Management**:
    *   **List View**: A clean, classic view for managing tasks.
    *   **Grid View**: A visual, card-based layout for a different perspective.
    *   **Kanban View**: A workflow visualization board with full drag-and-drop functionality.
*   **Powerful Task Operations**: Add, edit, complete, and delete tasks with instant feedback.
*   **Universal Search**: A real-time search bar that filters tasks instantly across all views.
*   **Advanced Filtering**: A collapsible sidebar to drill down and filter tasks by their status.
*   **Deep Personalization & Theming**:
    *   **Dark/Light Mode**: Switch between a stunning dark mode (default) and a clean light mode.
    *   **Accent Color Customization**: Personalize the entire UI with a selection of vibrant accent colors.
*   **Persistent State**: Your tasks, theme, view, and filter preferences are all saved to `localStorage`, so your workspace is always just how you left it.
*   **Interactive UI**: Smooth animations, satisfying micro-interactions, and visual feedback for every action.
*   **Responsive Design**: A fully responsive layout that looks and works great on all devices (conceptualized for mobile-first).

## 🛠️ Tech Stack

This project is built with a focus on a clean, modern, and dependency-free frontend stack.

*   **Frontend**: HTML5, CSS3, JavaScript (ES6+)
*   **Styling**: Vanilla CSS with Custom Properties (CSS Variables) for powerful theming. No pre-processors or frameworks.
*   **Icons**: [Lucide Icons](https://lucide.dev/) for a crisp and consistent look.
*   **Fonts**: [Inter](https://fonts.google.com/specimen/Inter) from Google Fonts for premium typography.
*   **Data Persistence**: All data is stored locally in the browser's `localStorage`. No backend is required.

## 🚀 Getting Started

Getting started with Stellar is incredibly simple. Since it's a pure frontend application, there's no build process or server setup required.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/stellar-app.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd stellar-app
    ```
3.  **Open `index.html` in your browser:**
    You can do this by simply double-clicking the `index.html` file in your file explorer, or by using a simple local development server for the best experience (this handles cross-origin requests if any were to be added in the future).

    **Using a simple Python server (if you have Python installed):**
    ```bash
    # For Python 3
    python -m http.server
    ```
    Then, open your browser and navigate to `http://localhost:8000`.

    **Using the VS Code Live Server extension:**
    If you are using Visual Studio Code, you can install the "Live Server" extension, right-click on `index.html`, and select "Open with Live Server".

## 🎨 Customization

Stellar is built from the ground up to be deeply customizable.

*   **Changing the Theme**: Use the sun/moon icon in the top-right corner of the application to toggle between dark and light modes.
*   **Changing the Accent Color**:
    1.  Click on your user avatar in the top-right corner.
    2.  Select "Settings" from the dropdown menu.
    3.  In the "Appearance" section, simply click on any of the color swatches to instantly change the application's accent color.

All of your appearance settings are automatically saved and will be applied the next time you open the app.
